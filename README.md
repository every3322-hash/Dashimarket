# 🍀 다시마켓 (Dashimarket)
**통합형 중고 & 자체 굿즈 거래 및 커뮤니티 플랫폼**

---

## 📌 프로젝트 기획 배경 및 개요 (Project Overview)

중고 거래 시장은 최근 환경 보호와 합리적인 소비 트렌드의 확산으로 빠르게 성장하고 있습니다. 
대표적인 중고 거래 플랫폼인 번개장터, 중고나라, 당근마켓 등을 분석하며 기존 서비스의 장점과 한계를 파악했습니다.

기존 중고 플랫폼들은 사용자 수가 많고 거래가 활발하다는 장점이 있지만, 광고가 많아 사용자 경험이 저하되거나 거래 외의 커뮤니티 기능이 제한적이라는 아쉬움이 있었습니다. 
또한 플랫폼 자체의 브랜드나 굿즈를 활용한 부가 수익 모델이 부족하다는 점도 확인할 수 있었습니다.

이러한 문제점을 개선하기 위해 본 프로젝트에서는 
광고를 최소화한 사용자 중심의 거래 환경, 
사용자 간 소통을 강화하는 커뮤니티 기능, 
플랫폼 자체 굿즈 판매를 통한 수익 구조 확보를 결합한 통합형 중고 거래 플랫폼을 기획하게 되었습니다. 
이를 통해 단순한 거래 서비스뿐 아니라 사용자 참여와 커뮤니티 활성화를 유도하고, 플랫폼의 지속적인 운영과 서비스 개선이 가능한 구조를 만들고자 했습니다.

## 💡 트러블슈팅
- 문제 <br>
채팅 기능을 플로팅 UI 방식으로 구현하면서 거래완료 버튼을 클릭해 다른 페이지로 이동할 경우 채팅 상태가 초기화되는 문제가 발생했습니다.

- 원인 <br>
채팅창을 별도의 페이지가 아닌 **플로팅 UI(JavaScript로 동적으로 생성되는 구조)**로 구현했기 때문에 페이지 이동이 발생하면 기존 DOM과 상태 정보가 모두 초기화되었습니다.
이로 인해 거래완료 처리  후 페이지가 변경될 때 채팅창의 상태와 데이터가 유지되지 않는 문제가 발생했습니다.

- 해결 <br>
거래완료 처리 이후에도 채팅 상태가 유지될 수 있도록 채팅 관련 데이터를 서버에서 다시 조회하도록 로직을 수정하고,
페이지 이동 후 채팅 목록과 상태 정보를 재요청하여 채팅 기능이 정상적으로 동작하도록 개선했습니다.

## 🌟 프로젝트 주요 서비스 기능 (Key Services)

### 1️⃣ 중고 상품 거래
- **상품 등록 및 관리**  
  중고 상품의 상세 정보, 상태, 카테고리, 가격 등을 등록 및 수정 가능
- **유저 간 채팅**  
  구매자와 판매자가 직접 소통하며 직거래 가능

### 2️⃣ 자체 굿즈 판매
- **굿즈 전용 상점**  
  프로젝트 자체 제작 상품(예: 캐릭터 쿠션 등)을 판매하는 독립적인 쇼핑 공간 제공
- **상품 관리 기능**  
  상품 등록, 수정, 재고 관리 등 운영자 중심 기능 지원

### 3️⃣ 유저 커뮤니티
- **자유 게시판**  
  모든 사용자가 자유롭게 소통하고 정보를 교환할 수 있는 게시판 제공
- **활동 참여 독려**  
  게시글 작성, 댓글, 좋아요 기능 등을 통해 사용자 활동 참여 촉진
